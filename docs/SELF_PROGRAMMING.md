# LivingCode 活性体 - 自我编程能力

## 当前已具备

| 能力 | 工具 | 状态 |
|------|------|------|
| 读文件 | read | ✅ |
| 写文件 | write | ✅ |
| 执行代码 | exec | ✅ |
| 动态加载 | exec (node/tsx) | ✅ |

## 自我编程架构

```
LivingCode 活性体
    │
    ├── 1. 感知需求 (用户输入/LLM)
    │
    ├── 2. 分析代码 (读取源文件)
    │   └── read: /agent/*.ts
    │
    ├── 3. 生成补丁 (LLM + Skills)
    │   └── code.solve / code.run
    │
    ├── 4. 应用修改 (写文件 + 重载)
    │   └── write + exec (重启服务)
    │
    └── 5. 验证效果
```

## 示例场景

### 场景1: 发现 Bug 自动修复
```
用户: "技能注册有问题，修复一下"
→ 读取 registry.ts
→ 分析问题
→ 生成修复代码
→ 写入文件
→ 重启服务
```

### 场景2: 添加新技能
```
用户: "添加一个天气查询技能"
→ 读取现有 skills 结构
→ 生成新技能代码
→ 写入文件
→ 热加载
```
