# LivingCode 生命体架构 v1.0

## 愿景

让 LivingCode/HuiFlow 成为一个有"生命"的 AI 助手：
- 有记忆（Memory）
- 能思考（LLM）
- 会行动（Skills/Tools）
- 能成长（Heartbeat/自动优化）

## 核心模块

### 1. 感知层 (Perception)

处理用户输入：
- 自然语言理解
- 意图识别
- 上下文提取

### 2. 思考层 (Cognition)

LLM 大脑：
- **Model**: MiniMax M2.5
- **Context**: System Prompt + Memory + History
- **Output**: 结构化 JSON 响应

### 3. 行动层 (Action)

执行Skills：
- `huiflow-api`: 调用业务 API
- `huiflow-sandbox`: 沙箱执行代码
- `huiflow-ui`: 生成 UI 指令

### 4. 记忆层 (Memory)

- **短期**: Session 历史（内存）
- **长期**: memory/YYYY-MM-DD.md（磁盘）
- **机制**: 每3条消息自动持久化

### 5. 心跳层 (Heartbeat)

后台守护进程：
```typescript
// 已注册任务
- cleanup_sessions: 每5分钟清理过期会话
- health_check: 每1分钟健康检查
```
